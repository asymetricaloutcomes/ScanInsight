# ScanInsight

Data intelligence platform for momentum traders — real-time market scanning, earnings intelligence, and AI-powered analytics.

## Features
- Real-time market breadth and distribution day tracking
- 5,000+ stock universe with RS ratings, ADR%, ATR$
- SEC 8-K earnings intelligence with AI classification (Claude Haiku)
- Episodic pivot detection
- TradingView Advanced Charts integration
- Live scanner with WebSocket data feed

## Stack
- **Backend:** Python, Flask, Supabase, DigitalOcean
- **Data:** Polygon/Massive, SEC EDGAR
- **Frontend:** JavaScript, TradingView Advanced Charts
- **AI:** Anthropic Claude Haiku
